자바스크립트 연산자

01. 산술연산자

산술연산자 종류 속성 설명
+ 더하기 연산을 할때 사용합니다.
문자와 문자, 문자와 변수 등을 연결할 때 사용합니다.
- 빼기 연산을 할 때 사용합니다.
* 곱하기 연산을 할 때 사용합니다.
/ 몫을 구 할 때 사용합니다.
% 나머지를 구할 때 사용합니다.
++ 1씩 증가시킬때 사용합니다.
-- 1씩 감소시킬때 사용합니다.

02. 대입연산자

대입 연산자 종류 설명
= 우변의 값을 좌변의 변수에 대입 합니다.
+= num = num + 값 과 같습니다.
-= num = num - 값 과 같습니다.
* num = num * 값 과 같습니다.
/ num = num / 값 과 같습니다.
% num = num % 값 과 같습니다.

03. 비교연산자

비교 연산자 종류 설명
< a < b 가 참이면 true, 거짓이면 false가 됩니다.
> a > b 가 참이면 true, 거짓이면 false가 됩니다.
<= a < b 가 참이면 true, 거짓이면 false가 됩니다
>= a > b 가 참이면 true, 거짓이면 false가 됩니다
== a == b 가 같으면 true, 않같으면 false가 됩니다.
== 는 데이터 타입은 비교하지 않습니다.
!= a == b 가 안같으면 true, 같으면 false가 됩니다.
=== a == b 가 데이터타입과 데이터가 같으면 true, 않같으면 false가 됩니다.
!== a == b 가 데이터타입 이나 데이터가 않같으면 true, 같으면 false가 됩니다.

04. 논리연산자

논리 연산자 종류 설명
&& (AND 연산자) a && b , a와 b모두 ture 일때 true가 됨니다.
|| (OR 연산자) a || b , a와 b 중 하나만 ture 일때 true가 됩니다.
모두 false 일때 false 됩니다.
! (NOT 연산자) a 가 true 이면 false, a가 false면 true로 바꾸어줍니다.
