HTML 기본 문서

html 요소는 <head>, <body> 요소로 구성되어있으며, <head> 요소 안에는 <meta>, <title>, <link>, <style>, <script> 등의 요소들이 올수 있습니다.

요소명 설명
<meta> HTML 문서의 설명, 키워드, 문서의 작성자 등을 지정합니다
<title> HTML 문서의 제목을 지정합니다.
<link> 외부의 CSS 파일을 HTML 문서에 연결할 때 사용합니다.
<style> HTML 문서 내에서 스타일(CS5)을 정의합니다.
<script> 자바스크립트를 HTML 문서 내부에서 정의하거나 외부에서 자바스크립트 파일을 불러올 때 사용합니다..

<html lang="ko">나 <meta charset="UTF-8> 처럼 태그 내부에 부가적인 설정값을 선언하는 것을 속성(attribute)라고 합니다.

lang 속성은 HTML 문서에서 사용하는 언어를 말하며 스크린 리더 에서 언서 설정에 따라 다르게 읽어 주기 때문에 HTML 문서에서 꼭 정의해야 하는 속성입니다.

<meta> 태그의 charset 속성의 UTF-8을 '문자 인코딩(charset)' 방식이라고 합니다. HTML의 문자 인코딩 방식으로 주소 사용되고 있습니다

<meta name = "vieport"/>

vieport 라고 하면 문서를 조절해 주는 속성이다.
